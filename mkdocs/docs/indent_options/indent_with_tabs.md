# indent_with_tabs

How to use tabs when indenting code.

Possible values:

* 0: Spaces only
* 1: Indent with tabs to brace level, align with spaces (default)
* 2: Indent and align with tabs, using spaces when not on a tabstop

Default is `1` (tabs).
